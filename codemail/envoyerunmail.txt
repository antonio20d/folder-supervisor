Ouvrez le fichier json : config.json
Modifiez les quatre paramètres
Les paramètres sont : 
	-L'adresse mail de l'expéditeur
	-Mot de passe
	-Le serveur d'envoi
	-Le port de ce serveur
	

Ouvrez le fichier : Envoyer_un_mail_avec_pièce_jointe.py
Rentrez les informations que vous souhaitez compléter.
	-Destinataire(s)
	-Objet
	-Texte
	-Le nombre de pièces jointes
	-Nom de la (ou des)pièce(s) jointe(s) 
	-Localisation

Pour mettre une pièce jointe, il faut renseigner le nom du fichier avec l'extension, exemple :

nom du fichier :image.png
adresse pièce jointe :D:\Images\image.png

Entrez

Le message est envoyé
